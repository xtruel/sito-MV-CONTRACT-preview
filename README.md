# M Contract S.r.l. — Official Website

Modern, responsive web portal for **M Contract S.r.l.**, specializing in turnkey solutions for Naval, Yacht, and Retail furniture.

## Features
- **Premium Design**: Modern dark theme with vibrant accents and glassmorphism.
- **Performance**: Built with Vite for ultra-fast loading and smooth interactions.
- **SEO Optimized**: Meta tags, OpenGraph, and semantic HTML for better visibility.
- **Responsive**: Fully optimized for mobile, tablet, and desktop devices.
- **Interactive**: Scroll-triggered animations and smooth navigation.

## Tech Stack
- [Vite](https://vitejs.dev/) - Frontend Tooling
- Vanilla JavaScript (ES6+)
- Vanilla CSS3 with Modern Variables
- Google Fonts (Outfit)

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Version 18 or higher recommended)

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`.

### Production Build
To create a production-ready bundle:
```bash
npm run build
```
The output files will be in the `dist` directory.

### Preview Production
```bash
npm run preview
```

## Deployment
This project is ready for deployment on **GitHub Pages**, **Netlify**, or **Vercel**.

### GitHub Pages
1. Ensure `vite.config.js` has `base: "./"`.
2. Build the project: `npm run build`.
3. Deploy the `dist` folder to the `gh-pages` branch.

### Netlify / Vercel
1. Connect your GitHub repository.
2. Set Build Command: `npm run build`.
3. Set Publish Directory: `dist`.
